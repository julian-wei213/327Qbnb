#  **Docker Run Commands**

To run the web-app
```
docker pull lobus/scrum_bucket_qbay:latest
docker run -p 8081:8081 lobus/scrum_bucket_qbay:latest python3 -m qbay
```

To run the entire system:
```
docker-compose up
```
